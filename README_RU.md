# Harmless Game SDDM Theme 🎲🔫

Язык: [English](README.md) | [Русский](README_RU.md)

Атмосферная тема экрана входа для SDDM в стиле подпольной игры ва-банк, вдохновленная визуалом **Buckshot Roulette**. Имеет чистый центрированный интерфейс, классическую типографику Garamond, полное отсутствие лишних зависимостей и палитру цветов, вытянутую прямо со стола Дилера.

Создана для того, чтобы вы подписывали отказ от ответственности и испытывали удачу при каждом запуске вашего Linux.

---

## ✨ Особенности

- 🎭 **Эстетика Buckshot Roulette:** Мрачная, глубокая палитра сигаретно-пороховых оттенков и ржавого игрового золота.
- 🎛️ **Кастомный селектор сессий:** Полностью кастомное центрированное меню без багов для быстрого выбора оконных менеджеров и окружений (Hyprland, Plasma, Sway и т.д.).
- 🖋️ **Классическая типографика:** Интегрированный изящный шрифт `Garamond` для воссоздания стиля винтажных бланков и документов.
- 🕒 **Живые часы:** Реальное отображение времени и локализованная дата.
- 👤 **Умный логин:** Автоматически подхватывает имя последнего пользователя или сбрасывает на дефолтного игрока (`PLAYER`).
- ⚡ **Максимальная легкость:** Нативный чистый код Qt6. Больше никаких тяжелых или отсутствующих в системе модулей графических эффектов.

## 📂 Установка

### 1. Ручная установка

Склонируйте репозиторий или скачайте ZIP-архив, после чего скопируйте папку в системную директорию тем SDDM:

```bash
# Клонирование репозитория
git clone https://github.com/neverloseagain1/Harmless-Game-SDDM-Theme

# Перенос темы в системную директорию с правильным именем папки
sudo cp -r Harmless-Game-SDDM-Theme/ /usr/share/sddm/themes/Harmless-Game
```
### 2. Активация темы

Отредактируйте конфигурационный файл SDDM (обычно это `/etc/sddm.conf` или `/etc/sddm.conf.d/theme.conf`), указав название темы в секции `[Theme]`:

```ini
[Theme]
Current=Harmless-Game
```

## ⚙️ Конфигурация

Вы можете легко изменить цвета, шрифты и фоновое изображение, отредактировав файл `theme.conf` внутри папки с темой:

```ini
[General]
# Цветовая палитра в формате HEX (Сукно стола и атмосфера игры)
ColorLight="#D4A359"
ColorDark="#120C0A"
ColorAccent="#C2913F"
ColorInputBg="#140F0A"

# Настройки шрифта
Font="Garamond"
FontSize="22"

# Путь к фоновому изображению
Background="backgrounds/background.jpg"
```

## 🛠️ Тестирование без перезагрузки

Вы можете проверить, как тема выглядит прямо сейчас в изолированном окне, не выходя из системы, выполнив команду в терминале:

```bash
sddm-greeter --test-mode --theme /usr/share/sddm/themes/Harmless-Game
```
